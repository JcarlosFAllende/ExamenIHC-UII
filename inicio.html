¿Cómo funciona?

Este sistema funciona con comandos de voz y con ayuda de una API de animación pueden darle vida las imagenes que tú desees agregar de manera local o con un URL de la imagen.

"Ayuda del sitio"

Al reconocer este comando de voz, se mandará a otra pestaña ofreciendo una pequeña guía de lo que hace el sistema.

"Animación"

Son 10 efectos en donde podrás visualizar a las imagines moviendose de una manera divertida

"Salir del sitio"

Al decir este comando se mostrará una ventana emergente diciendo si se quiere salir de la página.
